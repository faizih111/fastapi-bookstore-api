# 📚 Book Store API (FastAPI)

## 🚀 Overview

This is a simple Book Store REST API built using FastAPI.
The project demonstrates CRUD operations and backend development fundamentals, which are essential for building scalable systems.

## 🎯 Purpose

As a Computer Engineering student focusing on Machine Learning, I built this project to:

* Strengthen backend development skills
* Understand API design for ML systems
* Prepare for integrating ML models with real-world applications

## 🛠️ Tech Stack

* FastAPI
* Python
* Pydantic

## 📌 Features

* Create a new book
* Get all books
* Update book details
* Delete a book

## 📂 API Endpoints

* `GET /books` → Get all books
* `POST /books` → Add new book
* `PUT /books/{id}` → Update book
* `DELETE /books/{id}` → Delete book

## 🔮 Future Improvements

* Add database (SQLite/PostgreSQL)
* Implement authentication (JWT)
* Integrate ML model (e.g., book recommendation system)
* Deploy API

This project is a foundational step toward building ML-powered systems, where APIs are used to serve machine learning models in production.
